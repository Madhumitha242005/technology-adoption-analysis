# Amazon Product Review Analysis

## Project Overview

This project analyzes Amazon product review data to identify customer preferences, product ratings, pricing trends, discount patterns, and review behavior. The analysis is performed using Python in Jupyter Notebook with data visualization and statistical techniques.

## Dataset Information

The dataset contains Amazon product details and customer reviews.

### Features

* Product ID
* Product Name
* Category
* Actual Price
* Discounted Price
* Discount Percentage
* Rating
* Rating Count
* About Product
* User Information
* Review Details

## Project Phases

### Phase 1: Data Understanding

* Loaded dataset using Pandas
* Checked dataset structure
* Identified missing values
* Removed duplicate records
* Explored column distributions

### Phase 2: Descriptive Statistical Analysis

* Mean
* Median
* Standard Deviation
* Quartiles
* Percentiles

Analyzed:

* Rating
* Actual Price
* Discounted Price
* Discount Percentage

### Phase 3: Trend Investigation

Visualizations created:

* Category vs Average Rating (Line Chart)
* Correlation Heatmap
* Category vs Rating (Bar Chart)
* Actual Price vs Discounted Price Trend Plot
* Rating Count vs Rating (Scatter Plot)

### Phase 4: Correlation Analysis

Studied relationships between:

* Discount Percentage and Rating
* Rating Count and Rating
* Actual Price and Rating Count
* Actual Price and Discounted Price

Used:

* Correlation Matrix
* Heatmap Visualization

### Phase 5: Hypothesis Testing

Performed statistical analysis to evaluate relationships between ratings, prices, and discounts.

### Phase 6: Product Segmentation

Products were categorized based on:

* Rating Performance
* Discount Percentage
* Customer Engagement

### Phase 7: Business Insights

Key findings:

* Highly rated products attract more customer trust.
* Discount strategies influence customer purchasing behavior.
* Popular products receive a larger number of reviews.
* Pricing and discounting significantly affect product visibility.
